# Advertising Campaign Performance Dashboard

## Overview

This project presents an interactive Microsoft Excel dashboard for analyzing Amazon advertising campaign performance. The dashboard transforms raw campaign data into meaningful business insights using Pivot Tables, KPI analysis, and visual reporting.

## Tools Used

- Microsoft Excel
- Pivot Tables
- Dashboard Design
- Conditional Formatting
- KPI Reporting

## KPIs Analyzed

- Total Sales
- Click Through Rate (CTR)
- Return on Ad Spend (ROAS)
- Ad Spend Efficiency
- Campaign Performance
- Business Insights

## Dashboard Features

- Top Revenue Generators
- Highest ROAS Campaigns
- Cost Efficient Campaigns
- Low Performing Campaigns
- Interactive KPI Summary
- Visual Performance Dashboard

## Business Insights

- Compared campaign profitability across multiple advertising campaigns.
- Identified campaigns with the highest revenue and advertising efficiency.
- Highlighted campaigns requiring optimization due to poor performance.
- Enabled quick decision-making through an executive dashboard.
